MageTool by alistairstead

Magento command line tools

Additional ZF tools specifically for use during Magento development. Although Magento uses a great number of Zend Framework components and has a similar architecture to a Zend Framework application. It does not have any command line tools for use during development. These tools have been created to facilitate a number of repetitive tasks during development. Rather than switching between mysql tools and the Magento admin system you can run simple command and improve your workflow greatly

Using this Channel

This channel is to be used with the PEAR installer.

Registering the channel: 
pear channel-discover pear.magetool.co.uk
Listing available packages: 
pear remote-list -c magetool
Installing a package: 
pear install magetool/magetool
Installing a specific version/stability: 
pear install magetool/magetool-0.4.1
pear install magetool/magetool-beta
Receiving updates via a feed: 
http://pear.magetool.co.uk/feed.xml
